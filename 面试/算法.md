## [无重复字符的最长子串](https://leetcode.cn/problems/longest-substring-without-repeating-characters/description/)

#### 暴力解法

> 思路
> 1.‌ 枚举所有子串 ‌：用两层循环遍历字符串，外层循环确定子串的起始位置 i，内层循环确定子串的结束位置 j。
> 2.‌ 检查重复字符 ‌：对每个子串 s[i...j]，判断是否有重复字符。
> ‌3.记录最大值 ‌：若子串无重复，更新最大长度。

```javascript
function lengthOfLongestSubstringBruteForce(s) {
  let maxLength = 0;
  const n = s.length;

  // 遍历所有可能的子串起点 i
  for (let i = 0; i < n; i++) {
    // 用 Set 记录当前子串的字符
    const charSet = new Set();

    // 遍历所有可能的子串终点 j（从 i 开始）
    for (let j = i; j < n; j++) {
      const currentChar = s[j];

      // 如果字符已存在，说明当前子串有重复，跳出循环
      if (charSet.has(currentChar)) {
        break;
      }

      // 将字符加入 Set
      charSet.add(currentChar);

      // 更新最大长度
      maxLength = Math.max(maxLength, j - i + 1);
    }
  }

  return maxLength;
}
```

> ‌ 时间复杂度 ‌：O(n³) 空间复杂度：O(n)

#### 滑动窗口

> 思路
> 1.‌ 滑动窗口 ‌：用两个指针 left 和 right 维护一个窗口，表示当前子串范围。
> 2.‌ 哈希表记录位置 ‌：用哈希表（对象）存储每个字符 ‌ 最后出现的位置 ‌。
> 3.‌ 动态调整窗口 ‌：当发现重复字符时，更新 left 指针到重复字符的下一个位置，确保窗口内无重复。

```javascript
function lengthOfLongestSubstring(s) {
  const charIndexMap = {};
  let left = 0;
  let maxLength = 0;
  for (let right = 0; right < s.length; right++) {
    const currentChar = s[right];
    // 如果字符已存在，更新 left 指针
    if (
      charIndexMap[currentChar] !== undefined &&
      charIndexMap[currentChar] >= left
    ) {
      left = charIndexMap[currentChar] + 1;
    }
    // 更新字符最后出现的位置
    charIndexMap[currentChar] = right;
    // 更新最大长度
    maxLength = Math.max(maxLength, right - left + 1);
  }
  return maxLength;
}
```

> ‌ 时间复杂度 ‌：O(n) 空间复杂度：O(n)
