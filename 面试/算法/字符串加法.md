字符串加法的核心在于处理大数相加时可能出现的溢出问题，无法直接使用数值类型进行计算。因此，我们需要逐位相加并手动处理进位。以下是具体步骤：

核心 每次相加，从末尾开始逐位相加，同时处理进位。（倒着处理）

```
function addStrings(num1, num2) {
    let i = num1.length - 1; // 指向num1的末尾
    let j = num2.length - 1; // 指向num2的末尾
    let carry = 0; // 进位
    const result = []; // 保存结果的每一位

    // 从右到左逐位相加，直到所有位处理完毕且无进位
    while (i >= 0 || j >= 0 || carry > 0) {
        // 获取当前位的数字，若越界则视为0
        const digit1 = i >= 0 ? num1.charCodeAt(i) - 48 : 0;// 可以用Number，charCodeAt性能更好
        const digit2 = j >= 0 ? num2.charCodeAt(j) - 48 : 0;
        // 计算当前位的总和（包括进位）
        const sum = digit1 + digit2 + carry;
        // 更新进位
        carry = Math.floor(sum / 10);// 也可以用 parseInt(sum / 10),进位 需要与下一位相加
        // 当前位的值加入结果数组
        result.push(sum % 10);
        // 移动指针
        i--;
        j--;
    }

    // 反转数组并合并成字符串
    return result.reverse().join('');
}

```
